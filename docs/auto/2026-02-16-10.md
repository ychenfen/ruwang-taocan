# 自动/手工推进记录 - 2026-02-16 (10)

## 本次目标
- 按客户指定账单样式落地导出模板（卡号/入网日期/套餐/月租/状态/扶持期/稳定期/金额 + 总计）。

## 变更摘要
- 新增账单格式导出（后端）
  - 文件：`backend/src/routes/admin/reports.ts`
  - 新增接口：
    - `GET /admin/reports/bill.csv`
    - `GET /admin/reports/bill.xlsx`
  - 口径：
    - 卡号列追加归属标签：`(本人)` / `(团队)`
    - 入网日期转 `YYYY年M月`
    - 月租输出 `xx元`
    - 扶持期/稳定期按 `periodType` 自动分列（非当前期列为 `0%`）
    - 底部输出总计（`总计：x元`）
    - 默认排除 `ADJUSTMENT`，若传 `kind=ADJUSTMENT` 可单独导出调整口径。

- 账单导出纳入审计与预警
  - 文件：
    - `backend/src/routes/admin/reports.ts`
    - `backend/src/routes/admin/audit-logs.ts`
    - `backend/src/routes/admin/stats.ts`
  - 新增审计动作：`REPORT_EXPORT_BILL_FORMAT`
  - `audit-logs/export-summary` action 白名单补入该动作
  - `stats/alerts` 的导出频次异常统计补入该动作

- Admin 前端导出入口
  - 文件：`admin-web/app/(app)/reports/page.tsx`
  - 新增两个按钮：
    - `账单 CSV`
